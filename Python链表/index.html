Python 链表

Posted on 2022-08-31 Edited on 2023-01-18 In Study
Symbols count in article: 2k Reading time ≈ 2 mins.

链表的基本结构：

  • 节点：每个节点有两个部分，左边部分称为值域，用来存放用户数据；右边部分称为指针域，用来存放指向下一个元素的指针。

  • head：head节点永远指向第一个节点

  • tail：tail永远指向最后一个节点

  • None：链表中最后一个节点的指针域为None值

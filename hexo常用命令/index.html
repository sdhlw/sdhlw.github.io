Hexo常用命令

Posted on 2024-02-15 In Study
Symbols count in article: 404 Reading time ≈ 1 mins.

  • hexo clean : 清除缓存

  • hexo g : generate 生成页面

  • hexo s: server 启动本地服务预览, 将给出的链接复制粘贴到浏览器地址栏即可预览

  • hexo d: deploy 部署发布

创建文章：layout有三种：post(会发布)、draft(草稿)、page(首页标签页)。三种文章的模板文档在根文档夹的scaffolds里。命令：hexo n [layout] 文档名

  • hexo n 第一篇文章: 添加名为第一篇文章的发布文章（不加layout, 默认layout是post）
  • hexo n draft 第二篇文章: 添加名为第二篇文章的草稿文章，不发布
  • hexo P 第二篇文章: 草稿文章发布，即将文章从 source/_drafts 移动到 source/_posts
  • hexo n page 测试页：添加名为测试页的页面，页面不会出现在首页文章列表和归档中，也不支持分类和标签[1]，访问地址为：主页地址/测试页
